\doxysubsection{Подробное описание}
Класс для маршрутного (табличного) шифрования 

Реализует алгоритм маршрутного шифрования, который заключается в заполнении таблицы текстом по строкам и чтении его по столбцам в определенном порядке.

\begin{DoxyParagraph}{Принцип работы\+:}

\begin{DoxyEnumerate}
\item Текст записывается в таблицу построчно
\item Пустые ячейки заполняются пробелами
\item Для шифрования\+: чтение таблицы по столбцам снизу вверх справа налево
\item Для расшифрования\+: заполнение таблицы по столбцам и чтение построчно
\end{DoxyEnumerate}
\end{DoxyParagraph}
\begin{DoxyParagraph}{Пример шифрования\+:}

\begin{DoxyCode}{0}
\DoxyCodeLine{Текст: \textcolor{stringliteral}{"{}ПРИВЕТМИР"{}}, Колонки: 3}
\DoxyCodeLine{Таблица:}
\DoxyCodeLine{| П | Р | И |}
\DoxyCodeLine{| В | Е | Т |}
\DoxyCodeLine{| М | И | Р |}
\DoxyCodeLine{}
\DoxyCodeLine{Чтение для шифрования (по столбцам справа налево снизу вверх):}
\DoxyCodeLine{Столбец 3: Р, Т, Р → \textcolor{stringliteral}{"{}РТР"{}}}
\DoxyCodeLine{Столбец 2: И, Е, И → \textcolor{stringliteral}{"{}ИЕИ"{}}}
\DoxyCodeLine{Столбец 1: П, В, М → \textcolor{stringliteral}{"{}ПВМ"{}}}
\DoxyCodeLine{}
\DoxyCodeLine{Результат: \textcolor{stringliteral}{"{}РТРИЕИПВМ"{}}}

\end{DoxyCode}

\end{DoxyParagraph}
\begin{DoxyNote}{Заметки}
Алгоритм не является криптостойким и предназначен для учебных целей 
\end{DoxyNote}
