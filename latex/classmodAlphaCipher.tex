\doxysubsection{Подробное описание}
Класс для шифрования и расшифрования текста методом Гронсфельда 

Реализует алгоритм шифрования Гронсфельда для русского алфавита. Ключ устанавливается в конструкторе. Длина ключа может быть меньше текста, в этом случае ключ используется циклически.

Алгоритм\+:
\begin{DoxyEnumerate}
\item Каждой букве алфавита присваивается номер (0-\/32 для русского)
\item Текст и ключ преобразуются в числовые векторы
\item Выполняется сложение по модулю 33 (размер алфавита)
\item Результат преобразуется обратно в буквы
\end{DoxyEnumerate}

\begin{DoxyNote}{Заметки}
Для расшифрования используется тот же ключ 
\end{DoxyNote}
